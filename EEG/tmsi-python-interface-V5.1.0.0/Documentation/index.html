Welcome to the TMSi Python Interface documentation!

Welcome to the TMSi Python Interface documentation!#

Note

This project is under active development.

This documentation describes the TMSi Python Interface (TMSiPy), what it does, how it should work and what you can and cannot expect from TMSi regarding this interface. Please read it through carefully. The TMSi Python Interface is a library for Python written by TMSi to interface the SAGA and APEX Device Drivers to Python.

Warning

This documentation is written for the TMSi Python Interface V5.0.0.0 and higher. Please note that there have been major changes from V4.1.0.0 to V5.0.0.0 and higher. Please find the most important changes here to understand how to migrate your code from V4.1.0.0 to V5.0.0.0 and higher.

Getting started

Tutorials

Support

New to the TMSi Python Interface or interested to learn more? Please take a look at the Getting Started page to learn how to install the TMSi Python Interface, what is included and what examples you can follow to get acquainted with the Interface.

Several tutorials have been included in the documentation to learn how to use the TMSi Python Interface. Get started with learning about the Plotters, about performing EEG measurements with APEX or HD-EMG measurements with SAGA.

Do you have any questions about using the TMSi Python Interface? We cannot help you write your code, but any questions about using the TMSi Python Interface can be asked to your local distributor.
